# M10.L1
Idea de como será mi proyecto final sobre el cambio climático

Un bot de Discord para presentar ideas relacionadas con cómo inhibir o reducir el efecto del cambio climático en nosotros, los seres vivos. Dando ideas, consejos, mostrando videos o incluso los beneficios de tener un planeta mejor del que tenemos. Lo voy a utilizar con síntesis de voz para que sea más interactivo con el usuario, para dar sus ideas para que sean empleadas en el proceso final y también utilizar la visión por ordenador y así atraer al usuario a algo que pueda entender o interpretar a primera vista y así pueda comprender los beneficios y de qué forma aplicarlo para un mundo mejor.
